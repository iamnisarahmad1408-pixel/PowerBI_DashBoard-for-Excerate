# PowerBI_DashBoard-for-Excerate
This Power BI dashboard provides a comprehensive view of user engagement and opportunity allocation across the Excelerate platform. It is designed with an intuitive layout, interactive slicers, and visually rich analytics to help stakeholders monitor platform performance and applicant behavior.
Key Features & Insights
1. High-Level KPIs

The dashboard highlights essential platform metrics at a glance, including:

Total Opportunities: 20.30K

Team Allocated: 14.19K

Not Started: 1324

Started: 809

Dropped: 24

Rejected: 726

Withdrawn: 620

These KPIs help track the overall opportunity lifecycle and user progress.

2. Applicant Registration & Status Trends

A multi-line trend chart visualizes:

Monthly applicant registrations

Status progression (Applied, Started, Dropped, Rejected, etc.)

This provides visibility into seasonal engagement patterns and platform usage fluctuations.

3. Status Distribution

A tree map shows how applicants are distributed across various status categories such as:

Team Allocated

Rewards Awarded

Started

Rejected

This helps stakeholders quickly identify dominant categories and potential bottlenecks.

4. Most Sought Opportunities

A horizontal bar chart displays the most popular opportunity areas, including:

Data Visualization (5.7K)

Digital Marketing

Health Care

Innovation & Entrepreneurship

Project Management

This reveals user interest trends and demand across different fields.

5. Preferred Opportunity Categories

A column chart breaks down preferred opportunity types such as:

Internships

Events

Engagements

Courses

Competitions

This helps identify which categories attract the highest participation.

6. Interactive Filters

Users can slice the data using:

Country

Status Description

Gender

These filters enable personalized, targeted analysis based on demographic factors.

Overall Design & Usability

Clean, soft-toned theme with consistent branding

Buttons for navigation (Platform Activity, Global Reach, Demographic, Skills Development)

Clear visual hierarchy for easy insights

Ideal for management reporting, program monitoring, and performance analytics
